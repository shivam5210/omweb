## 🌟 Overview

**Dressify** is a cutting-edge e-commerce platform that revolutionizes online fashion shopping. Using advanced virtual try-on technology, customers can see exactly how clothes look on them before making a purchase decision. This reduces return rates, increases customer satisfaction, and transforms the online shopping experience.

## ✨ Key Features

### 👕 Advanced Virtual Try-On System
- **Interactive 3D Avatar** with realistic clothing visualization
- **Real-Time Updates** - See clothing changes instantly
- **Mix & Match** - Combine tops, bottoms, and shoes
- **Detailed Product Info** - Material, ratings, and descriptions

## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection for loading placeholder images

### Installation

1. **Clone the Repository**
   ```bash
   git clone https://github.com/shivam5210/omweb.git
   cd omweb
   ```

2. **Open in Browser**
   ```bash
   # Simply open the index.html file
   open index.html
   ```

### No Setup Required!
- ✅ No node.js installation needed
- ✅ No build process
- ✅ No external dependencies
- ✅ Works completely offline (except for placeholder images)

## 📖 How to Use

### Virtual Try-On
1. **Browse Products** - Scroll through the product collection
2. **Click to Try** - Select any clothing item
3. **Watch Avatar Update** - See the item on the virtual avatar
4. **Mix & Match** - Try different combinations
5. **View Details** - Check material, ratings, and price

### Shopping
1. **Featured Collections** - View trending items
2. **Add to Cart** - Click "Add to Cart" button
3. **Track Cart** - See cart count in navigation
4. **Filter by Category** - Use tabs to browse by type
