# 👕 ClothFit - Virtual Try-On E-Commerce Website

A modern e-commerce platform with virtual clothing try-on technology that allows customers to see how clothes look on an avatar before purchasing.

## Features

### 🎨 Virtual Try-On System
- Interactive avatar that displays selected clothing items
- Real-time visualization of how clothes look together
- Support for:
  - **Tops** (T-shirts, Polos, Hoodies, V-necks)
  - **Bottoms** (Jeans, Pants, Chinos, Skirts, Shorts)
  - **Shoes** (Sneakers, Formal Shoes, Loafers, Athletic Shoes, Sandals)

### 💳 Shopping Features
- Product catalog with 15+ items
- Browse by category (All, Tops, Bottoms, Shoes)
- Product filtering and selection
- Add to cart functionality
- Featured products section

### 🎯 User Experience
- Responsive design (works on mobile, tablet, desktop)
- Modern gradient UI with smooth animations
- Sticky navigation bar
- Professional product cards and featured section
- User-friendly product information display

## Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with Flexbox and Grid
- **Vanilla JavaScript** - No dependencies, pure functionality

## How to Use

1. **Clone the repository:**
   ```bash
   git clone https://github.com/shivam5210/omweb.git
   ```

2. **Open in browser:**
   - Simply open `index.html` in your web browser
   - No server or build process required

3. **Virtual Try-On:**
   - Click on any clothing item in the product grid
   - Watch the avatar update with your selection
   - Mix and match tops, bottoms, and shoes
   - View product details and prices

4. **Shopping:**
   - Click "Add to Cart" on featured products
   - Selected items appear in the cart counter
   - Browse different categories using filter tabs
